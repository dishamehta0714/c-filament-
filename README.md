📘 C String Programs 
This project contains two basic C programs related to string operations:

Frequency of each character in a string

Palindrome string check

These programs help in understanding string handling, loops, and conditions in C.

🧮 Program 1: Character Frequency

https://onlinegdb.com/Prst65z5H

🔹 Description

This program counts how many times each character appears in a given string.

🔹 How it Works

Takes input string from user

Compares each character with others

Marks counted characters as '0' to avoid repetition

Prints frequency of each character

🔹 Example
Input: hello

Output:
h=>1
e=>1
l=>2
o=>1

🔁 Program 2: Palindrome Check

https://onlinegdb.com/wV44Vco1Tn

🔹 Description

This program checks whether a string is a palindrome or not.

👉 A palindrome is a word that reads the same forward and backward.

🔹 How it Works

Finds length using strlen()

Compares characters from start and end

If all match → palindrome

If any mismatch → not palindrome

🔹 Example
Input: madam
Output: the string is a palindrome.

Input: hello
Output: the string is not a palindrome.
⚙️ Requirements

C Compiler (e.g., GCC)

Basic knowledge of C programming
